## INSTALLATION

Open terminal:

    cd ~/Library/Application Support/Colloquy
    mkdir Repositories
    cd Repositories
    git clone git@github.com:benlinton/colloquy-theme-campfire.git
    cd ../Styles
    ln -s ../Repositories/colloquy-theme-campfire/Campfire.colloquyStyle ./Campfire.colloquyStyle

Restart Colloquy.
